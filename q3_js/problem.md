# Q3：JavaScript（最小分析UI）

- 制限時間目安：1時間
- テーマ：閾値操作で集計結果が変わるミニUI（状態管理 + 描画更新）

## 問題

`index.html` 1ファイルのみで、以下の要件を満たすページを作成せよ。

- スコア配列（数値配列）を JavaScript 内に持つ（例：`[12, 40, 51, ...]`）
- 以下のボタンを持つ
  - 「閾値 +1」
  - 「閾値 -1」
  - 「リセット」
- 画面に以下を表示する
  - 現在の閾値
  - `score >= 閾値` を満たす件数
- 閾値は `0〜100` の範囲にクリップする（それ未満/超過にならない）

## 採点観点（簡易）

- 状態（threshold）と表示（render）を分離している
- ボタン操作時に即座に表示が更新される
- 境界（0未満、100超過）で壊れない
