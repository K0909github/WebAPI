# Q1：アルゴリズム（AtCoder A〜B）

- 制限時間目安：1時間
- テーマ：時系列の制約付き最長区間（尺取り法 / Two Pointers）

## 問題

長さ $N$ の非負整数列 $A_1,\dots,A_N$ と整数 $K$ が与えられる。

連続部分列 $A_l,\dots,A_r$ の総和が $K$ 以下となるような区間のうち、最長の長さを求めよ。

## 制約

- $1 \le N \le 2\times 10^5$
- $0 \le A_i \le 10^9$
- $0 \le K \le 10^{18}$

## 入力形式

```
N K
A1 A2 ... AN
```

## 出力

条件を満たす連続部分列の最長の長さ（整数）を出力せよ。

## サンプル

入力:
```
6 7
2 1 3 2 4 1
```

出力:
```
3
```

## 補足

- $A_i$ は非負なので、尺取り法で $O(N)$ が可能です。
- $O(N^2)$ は間に合いません。
